



# TweetSaver - X (Twitter) Video Downloader

A modern, responsive web app to download videos, GIFs, and images from X (Twitter) posts.

## Features
- Paste a tweet URL, get downloadable media
- Multiple quality options (video)
- Dark/light theme toggle
- Mobile‑friendly hamburger navigation (grouped next to theme toggle)
- Ad and affiliate banner placeholders (ad at the bottom)
- Fully responsive, premium UI

## Project Structure
- `index.html` – main downloader
- `about.html`, `contact.html`, `privacy.html`, `terms.html` – static pages
- `style.css` – styling with theme support
- `script.js` – client-side validation, theme, mobile nav, AJAX
- `app.py` – Flask backend
- `requirements.txt` – Python dependencies
